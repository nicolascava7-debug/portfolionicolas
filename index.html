PROJECT: LOOP_LEGACY
// section_01 — le brief
Le Brief
Le studio indépendant Devolver Digital, reconnu pour ses jeux audacieux à l'identité forte, lance son nouveau projet ambitieux Loop Legacy. Ce jeu propose un concept où le joueur incarne un héros ou une héroïne pris dans une boucle temporelle, dans une mégalopole futuriste : Nox. Dans Loop Legacy, le héros Kortex ou l'héroïne Karma se bat pour briser une boucle temporelle qui piège Nox. Chaque jour se répète, mais les décisions du joueur modifient subtilement chaque boucle.
Le projet consiste à créer une identité visuelle marquante pour ce nouveau jeu vidéo. Les livrables sont la réalisation d'un moodboard, d'une palette de couleur, d'une typographie et d'un logo typographique, et d'une affiche pour le jeu.

// section_01 — le brief
Le Brief
Dans le cadre d'un projet scolaire, nous étudions le festival We Love Green, événement précurseur alliant programmation musicale et engagements écoresponsables. L'agence Fest'Event, spécialisée en digitalisation événementielle, a été mandatée pour concevoir une extension numérique interactive. L'enjeu n'est pas de substituer l'expérience physique, mais de l'enrichir via un univers digital inédit.
Problématique : Comment offrir à la Gen Z une expérience de festival virtuelle aussi immersive et engagée que l'événement physique, sans les contraintes de déplacement et avec un impact numérique réduit ?
